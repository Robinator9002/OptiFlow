Installation

Herunterladen und Starten

Um den Dateimanager zu installieren, laden Sie das komplette Programm inklusive des .venv/-Ordners herunter. Dieser Ordner enthält alle notwendigen Dateien, einschließlich der .json-Konfigurationsdateien, des Backends (Python/FastAPI) und des Frontends (React).

Nach dem Herunterladen führen Sie die start.bat-Datei aus. Diese Datei startet den Server mit Uvicorn und anschließend die Webanwendung. Alternativ können Sie auch die start_server.bat-Datei verwenden, um nur den Server zu starten, oder die start_app.bat-Datei, um nur die Webanwendung zu starten.

  1. Laden Sie das Programm herunter.
  2. Entpacken Sie die heruntergeladene Datei.
  3. Navigieren Sie zum entpackten Ordner.
  4. Führen Sie die start.bat-Datei aus.
  5. Warten Sie, bis sowohl der Server als auch die Webanwendung gestartet sind.

Nutzung der Anwendung

Nachdem der Server und die Webanwendung gestartet sind, können Sie die Anwendung in Ihrem Webbrowser nutzen. Die Anwendung kommuniziert automatisch mit dem Server über Axios, um Anfragen zu senden und Daten zu empfangen.

Stellen Sie sicher, dass Ihr Webbrowser JavaScript aktiviert hat, da die Anwendung dies für die Kommunikation mit dem Server benötigt.

Voraussetzungen

Um den Dateimanager nutzen zu können, benötigen Sie:

  • Einen modernen Webbrowser (z.B. Chrome, Firefox, Edge).
  • Ein Betriebssystem, das die Ausführung von .bat-Dateien unterstützt (z.B. Windows).
  • Python 3.7 oder höher (falls Sie den Server manuell starten möchten).
