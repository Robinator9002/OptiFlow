Installation und Start

Die Inbetriebnahme des OptiFlow Dateimanagers ist dank eines komfortablen Installationsprogramms für Windows sehr einfach. Für andere Betriebssysteme oder Entwickler steht eine manuelle Methode zur Verfügung.

Methode 1: One-Click-Installer (Empfohlen für Windows)

Dies ist der einfachste und schnellste Weg, OptiFlow auf einem Windows-System (Server oder Workstation) zu installieren. Der Installer ist ca. 300 MB kleiner als das manuelle Paket und übernimmt alle Konfigurationsschritte für Sie.

  1. Installer herunterladen: Laden Sie die bereitgestellte Installer-Datei (OptiFlow-Dateimanager-Setup.exe) herunter.
  2. Installer ausführen: Führen Sie die Datei mit einem Doppelklick aus. Die Windows-Benutzerkontensteuerung fragt möglicherweise um Erlaubnis, die Sie bestätigen müssen.
  3. Installation abschließen: Das Programm wird nun automatisch an den Standard-Speicherort installiert. Der Installer erledigt alles Weitere im Hintergrund.
  4. Verknüpfungen nutzen: Nach der Installation finden Sie automatisch eine Verknüpfung auf Ihrem Desktop und/oder im Startmenü.

Anwendung starten

Nutzen Sie einfach die neu erstellte Desktop- oder Startmenü-Verknüpfung, um OptiFlow zu starten. Es ist kein manueller Start von Skripten mehr notwendig. Der Server und die Applikation starten automatisch im Hintergrund.

Methode 2: Manuelle Installation (Für Linux & erfahrene Windows-Nutzer)

Diese Methode gibt Ihnen volle Kontrolle über den Installationsort und ist für Linux-Systeme sowie für Entwickler oder Administratoren auf Windows gedacht, die eine portable Version bevorzugen.

Schritt 1: ZIP-Paket herunterladen

Laden Sie das ZIP-Paket (OptiFlow-Source.zip) herunter und entpacken Sie es an einem permanenten Ort auf Ihrem System (z.B. C:\OptiFlow\ unter Windows oder /opt/optiflow/ unter Linux).

Schritt 2: Anwendung starten

  • Unter Windows: Führen Sie die Datei start.bat per Doppelklick aus.
  • Unter Linux: Öffnen Sie ein Terminal, navigieren Sie in das entpackte Verzeichnis und führen Sie das Skript mit ./start.sh aus. (Möglicherweise müssen Sie es zuerst mit chmod +x start.sh ausführbar machen).

In beiden Fällen startet nun der Backend-Server in einem Konsolenfenster, das geöffnet bleiben muss. Die Anwendung selbst öffnet sich in einer eigenen Electron-Umgebung, ein separater Browser ist nicht zwingend notwendig.

Wo werden meine Daten gespeichert?

Der Speicherort für Ihre Daten (Benutzerprofile, Suchindex etc.) hängt von der Installationsmethode ab. Dies ist wichtig für Backups oder falls eine manuelle Bereinigung notwendig sein sollte.

Bei Installation mit dem Installer:

Der Datenordner befindet sich im Anwendungsverzeichnis unter backend/data. Der vollständige Pfad lautet normalerweise:
C:\Users\<IhrBenutzername>\AppData\Local\frontend\backend\data

Bei manueller Installation (ZIP/Source):

Der Datenordner befindet sich direkt im Hauptverzeichnis, in das Sie die Anwendung entpackt haben.
Beispiel: C:\OptiFlow\data oder /opt/optiflow/data

Nächster Schritt

Die Anwendung läuft! Nun ist es an der Zeit, sich zum ersten Mal anzumelden und das System zu konfigurieren.
