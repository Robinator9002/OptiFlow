Die Anwendung anpassen: Ein Überblick

Der Einstellungs-Tab (erreichbar über den Button ⚙️ Einstellungen oben rechts) ist die zentrale Anlaufstelle, um OptiFlow an Ihre persönlichen Vorlieben und die Anforderungen Ihres Unternehmens anzupassen. Die Einstellungen sind in verschiedene Reiter unterteilt, die wir hier im Detail erläutern.

Speichern nicht vergessen!

Nachdem Sie Änderungen vorgenommen haben, müssen Sie unten auf den "Speichern"-Button klicken, damit diese wirksam werden. Ein Verlassen der Einstellungen ohne Speichern verwirft alle Änderungen.

Allgemein

Passen Sie hier das Erscheinungsbild der Anwendung an. Diese Einstellungen sind benutzerspezifisch.

  • Erscheinungsbild (Theme): Wählen Sie zwischen einem hellen Standard-Design, einem augenschonenden Dunkelmodus und einem Modus mit hohem Kontrast für bessere Lesbarkeit.
  • Barrierefreiheit (Schriftgröße): Skalieren Sie die Schriftgröße der gesamten Anwendung stufenlos über einen Schieberegler, um sie an Ihre Bildschirmauflösung und Sehstärke anzupassen.

Suche (Admin)

Konfigurieren Sie hier das Verhalten der globalen Suche für alle Benutzer.

  • Maximale Suchergebnisse: Legen Sie fest, wie viele Treffer maximal in der Ergebnisliste angezeigt werden sollen, um die Übersicht zu wahren.
  • Relevanz-Scores anzeigen: Blenden Sie neben jedem Suchergebnis den numerischen Relevanz-Score ein, damit Benutzer die Sortierung besser nachvollziehen können.
  • Punktesystem anpassen: Verändern Sie die Gewichtung für Suchtreffer. Sie können festlegen, wie viele Punkte ein exakter Treffer im Dateinamen, ein teilweiser Treffer oder ein Treffer im Inhalt erhält.
  • Snippet-Größe: Passen Sie an, wie viele Zeichen vor und nach einem Treffer in der Snippet-Vorschau der In-Datei-Suche angezeigt werden.

Scanner (Admin)

Hier steuern Administratoren die Details des Indexierungs-Prozesses. Dies ist eine der wichtigsten Konfigurationsseiten.

Definieren Sie Zielordner, Dateitypen und weitere Scan-Parameter.

  • Zielordner: Legen Sie die Startverzeichnisse für den Scan fest. Über den Button "Neuer Ordner" öffnet sich der interaktive Folder Selector, mit dem Sie bequem durch die Server-Verzeichnisse navigieren und Ordner zur Liste hinzufügen können.
  • Dateierweiterungen: Bestimmen Sie, welche Dateitypen (z.B. .pdf, .docx, .txt) überhaupt indexiert werden sollen. Sie können eine der vordefinierten Gruppen wählen oder eine benutzerdefinierte Auswahl treffen.
  • Inhalt indexieren / PDF Konversion: Legen Sie fest, ob auch die Inhalte der Dateien (und nicht nur die Namen) in den Index aufgenommen werden sollen. Aktivieren Sie die PDF-Konversion, damit auch Text aus PDFs extrahiert wird.
  • Maximale Dateigröße (KB): Schließen Sie sehr große Dateien vom Scan aus, um Ressourcen zu sparen.
  • Weitere Optionen: Legen Sie die maximale Anzahl an CPU-Kernen für den Scan fest oder definieren Sie eine Scan-Verzögerung zur Schonung älterer Systeme.

Benutzer

Verwalten Sie Ihr eigenes Konto oder (als Admin) alle Konten im System.

Für alle Benutzer:

  • Benutzernamen ändern: Passen Sie Ihren Anzeigenamen an.
  • Passwort ändern: Aus Sicherheitsgründen sollten Sie Ihr Passwort regelmäßig ändern. Hier können Sie nach Bestätigung Ihres alten Passworts ein neues vergeben.
  • Konto löschen: Löschen Sie Ihr eigenes Konto (erfordert Passwortbestätigung).

Zusätzlich für Admins:

Admins haben die volle Kontrolle über alle Benutzerkonten.

  • Benutzerliste anzeigen: Sehen Sie eine Liste aller registrierten Benutzer inklusive ihres Admin-Status und des letzten Logins.
  • Benutzer hinzufügen: Erstellen Sie neue Benutzerkonten und legen Sie deren initiales Passwort sowie den Admin-Status fest.
  • Passwort zurücksetzen: Setzen Sie das Passwort für einen Benutzer zurück (erfordert Ihr Admin-Passwort).
  • Admin-Status ändern: Erteilen oder entziehen Sie anderen Benutzern Administratorrechte.
  • Benutzer löschen: Entfernen Sie ein Benutzerkonto aus dem System (erfordert Ihr Admin-Passwort).

System (Admin)

Globale Einstellungen, die das Verhalten des Servers beeinflussen.

  • Zeitgesteuerte Events: Ein mächtiges Werkzeug zur Automatisierung. Erstellen Sie Events, die zu bestimmten Zeiten (täglich, wöchentlich, stündlich) ausgeführt werden. So können Sie z.B. einen nächtlichen Index-Scan oder eine wöchentliche OCR-Konvertierung einrichten, ohne manuell eingreifen zu müssen.
  • Server herunterfahren: Fährt den OptiFlow Backend-Server sicher herunter. Erfordert eine doppelte Bestätigung mit Ihrem Admin-Passwort. Die Anwendung ist danach nicht mehr erreichbar und muss manuell via Installer-Verknüpfung oder start.bat neu gestartet werden.
